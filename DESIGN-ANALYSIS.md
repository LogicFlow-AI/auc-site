# WordPress Design Analysis

## Site: adventist.org.au

### Color Palette (Extracted)

**Primary Colors:**
- Primary Blue: `#0066cc` (links, buttons, accents)
- Dark Gray: `#333333` (text, headers)
- Orange Accent: `#dd6925` (seen in some UI elements)
- Light Gray: `#f5f5f5` (backgrounds)
- Border Gray: `#e0e0e0` (borders, dividers)

**Text Colors:**
- Primary Text: `#333333`
- Secondary Text: `#666666`
- Light Text: `#999999`
- White: `#ffffff`

### Typography

**Font Family:**
- Primary: System fonts (Times, serif fallback visible)
- Likely using: Inter, system-ui, or similar sans-serif

**Font Sizes:**
- Base: 16px
- Headings: Various sizes (h1: ~36-48px, h2: ~30px, h3: ~24px)

### Layout Structure

**Header:**
- Logo on left: "Australia's Church of Seventh-day Adventists"
- Navigation menu: Horizontal, right side
- Menu items: Our Team, News, Events, Careers, Ministries, Education, Health, Media, Statements of Belief
- Search icon on right
- Mobile menu toggle

**Homepage Sections:**
1. Hero section with video/background
2. Main heading: "Bible Study, Daily Devotional and More..."
3. Three-step process section (Bible Study steps)
4. Quote/testimonial section
5. "Find a Church Near me?" section with map and conference links
6. FAQ section with accordion-style questions
7. Footer with multiple columns

**Footer:**
- Four columns:
  1. Quick Links
  2. Conferences (by region)
  3. Resources (Adventurers, Pathfinders, Schools, etc.)
  4. Media (Faith FM, Hope Channel, etc.)
- Social media icons (Facebook, Instagram, YouTube)
- Copyright and links (Site Map, Privacy Policy, Contact, Careers)
- "Website by" credits

### Component Styles

**Buttons:**
- Primary: Blue background (#0066cc), white text
- Secondary: White background, blue text, border

**Cards:**
- White background
- Shadow on hover
- Rounded corners

**Navigation:**
- Horizontal menu
- Hover states
- Mobile hamburger menu

### Spacing

- Container max-width: ~1200px (typical)
- Section padding: ~48-64px vertical
- Card padding: ~24px
- Grid gaps: ~24-32px

### Breakpoints

- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px
